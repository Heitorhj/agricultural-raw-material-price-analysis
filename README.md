# 📊 Agricultural Raw Material Price Analysis

## 📖 Sobre o projeto

Este projeto consiste em uma **Análise Exploratória de Dados (EDA)** utilizando Python para estudar a evolução dos preços de diferentes matérias-primas agrícolas ao longo do tempo.

Durante a análise foram realizadas etapas de limpeza dos dados, tratamento de valores ausentes, análise estatística e construção de visualizações para identificar padrões, distribuições e correlações entre as variáveis.

---

## 🎯 Objetivos

* Explorar o conjunto de dados.
* Realizar o tratamento dos dados.
* Identificar valores ausentes e inconsistências.
* Analisar a distribuição dos preços.
* Avaliar a variação percentual das matérias-primas.
* Investigar a correlação entre as variáveis.
* Produzir visualizações que auxiliem na interpretação dos dados.

---

## 📂 Dataset

O conjunto de dados contém informações históricas sobre preços de diversas matérias-primas agrícolas, incluindo suas respectivas variações percentuais ao longo do tempo.

Algumas das commodities analisadas:

* Copra
* Rubber
* Cotton
* Coarse Wool
* Fine Wool
* Hard Log
* Soft Log
* Wood Pulp
* Plywood
* Hide
* Hard Sawnwood
* Soft Sawnwood

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Heitorhj/agricultural-raw-material-price-analysis.git
```

Entre na pasta do projeto:

```bash
cd agricultural-raw-material-price-analysis
```

Instale as dependências:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

Abra o notebook:

```bash
jupyter notebook
```
